# Project Overview
This Power BI project focuses on building an interactive, insight-driven dashboard to support business decision-making.
The dashboard transforms raw data into clear KPIs, trends, and actionable insights using industry-standard BI practices.

# Objectives:
1. Convert raw data into meaningful business insights
2. Track key performance indicators (KPIs)
3. Enable data-driven decisions through interactive visuals
4. Apply best practices in data modeling and DAX


# Tools & Technologies: 
1. Power BI Desktop
2. Power Query (Data Cleaning & Transformation)
3. DAX (Measures & Calculations)
4. Data Modeling (Star Schema)

# Key Features :
1. Interactive KPIs & summary cards
2. Trend analysis using line & bar charts
3. Category-wise and time-based insights
4. Clean, professional BI color theme

#  Dashboard Highlights :
1. Actual vs. Budget performance overview
2. Income and Expense: Actual vs. Budget Yearly analysis
3. Category/segment comparison
4. Drill-down enabled visuals

# How to Use :
Download the .pbix file > Open using Power BI Desktop > Refresh data if required > Interact with slicers and visuals to explore insights
